# Content Polish Workflow

Status: active

Use this workflow for copy and layout clarity work.

Steps:

1. Review the relevant page or content file.
2. Make small edits only.
3. Preserve routes and app behavior.
4. Build if source files changed.
5. Update output.md with a short result.
6. Update pointer.md and run-log.md.

Acceptance:

The content is clearer without breaking routes.
