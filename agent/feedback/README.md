# Feedback Folder

Status: active

This folder stores repo-local feedback for long-running Lost Pages work.

Use it to keep important direction out of chat-only memory.

## Files

- `how-to-feedback.md` explains the feedback process.
- `active-feedback.md` stores current feedback that still matters.
- `processed-feedback.md` stores feedback that has been addressed.
- `archive/` stores old feedback when the active file gets too long.

## Rule

Do not treat every sentence as permanent. Promote clear, repeated, or explicit feedback into active feedback.
